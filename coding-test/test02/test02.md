        이중for문 방식을 채택하였다.

        처음엔 배열 내 동일한 인덱스 두개를 합쳐도 되는 줄 알고 진행했다가 
        정답과 다른 것을 보고 문제를 다시 보니 서로 다른 인덱스의 두 수를 뽑아야 하는 것을 확인하고
        이중 for문 안에 if ( i != j) 조건을 추가하였다. (문제를 잘 파악하자! 요구사항 명세를 잘 읽자!!)
